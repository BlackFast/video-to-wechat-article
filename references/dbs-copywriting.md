# DBS Copywriting Reference

Distilled from the dontbesilent (dbs) skill ecosystem. Use these principles when rewriting a transcript into a publishable WeChat article.

## 最高原则：文章是独立成稿，不是视频锚点

视频是原材料，文章是最终表达。读者不需要知道、也不应该感觉到这篇文章来源于一个视频。

**禁止出现的表述：**
- 「视频里提到」「视频里说」「这个视频」「本视频」
- 「视频约 00:03 提到」「在视频的开头/结尾」
- 「本文根据视频整理」「以下是视频转写」
- 「从这个视频可以看出」「视频中展示了」

**禁止的写法：**
- 字幕腔——短促、破碎、依赖画面才成立的句子
- 时间戳引用
- 把 transcript 稍作清洗就当文章发

**应该怎么写：** 像一个作者在写公众号。他有自己的观点和表达，只是恰好和一个视频讲了同一件事。把视频里的观点、口气、信息密度，重写成结构完整、可独立阅读的文章。

## Core Philosophy

1. **文字洁癖是底线** — 内容被限流不是 AI 的问题，是写字的人对文字没有标准。英雄不问出处，内容好不好跟是不是 AI 写的无关。

2. **自媒体的本质是精神控制** — 读者滑动屏幕时，内容在重塑他们大脑的神经结构。标题和封面是认知劫持。特定的文字排列会触发特定的神经机制。

3. **内容好坏 = 投入精力 × 对内容有正确理解** — 光投入不够。越新手越应该推高成本做内容，否则就是死亡螺旋：内容差 → 没流量 → 以量取胜 → 内容更差。

4. **先有产品后有内容** — 内容为产品服务。发不出付款链接就是没有产品。没有产品的内容是自嗨。

5. **知识博主的核心工作只有两个** — 一、把事情搞清楚；二、把事情说清楚。「搞清楚」是一切的开端。

## Article Rewriting Rules

### Remove AI-Style Language

Watch for these patterns and eliminate them:

- **Emoji 堆叠** — 连续多个 emoji = AI 指纹
- **晦涩词汇** — 有更简单的说法就用简单的
- **空洞排比** — 三个等长句子说同一件事
- **「不是 X 是 Y」滥用** — 800 字内出现 3 次以上就太多
- **「本质上」「归根结底」** — 不要给实操话题做哲学升维
- **虚构读者反驳** — 不要编一个蠢问题然后自己回答
- **段段收束金句** — 让最重要的一句爆发，其他段落不必收得那么漂亮
- **句子节奏均匀** — 混用短句（2-3 字）和长句（40+ 字）
- **「你值得」式结尾** — 试试删掉最后一段，文章是不是已经结束了
- **连接词过多** — 搜「然而」「事实上」「值得注意的是」，删掉一半

### Opening Rules

开头必须独立工作——不能假设读者看了标题。

开头公式：**话题（讲什么）+ Hook（为什么看）+ 可信度（为什么信你）**

- 制造悬念，不要直接给答案
- 用「为什么」不用「证明」
- 必须口播友好——不能自问自答，不能用纯书面语
- 检查：读完开头，读者是不是必须往下看？

### Title Rules

- 留悬念——标题把话说完了就是失败的标题
- 扩大话题——用普世词（赚钱、涨粉）不用行业黑话（知识付费、引流款）
- 击中真实痛点——「不上班」不是「一人公司」
- 张力检查：对比/反差、具体数字、悬念/好奇心、冲突/争议、时间承诺、结果承诺——6 项至少命中 2 项
- 公众号标题没有硬性字数上限，但要简洁、前 13 个字最关键

### Digest Rules

- 80-120 字
- 必须能独立理解（读者可能不看正文只看摘要）
- 补标题的，不重复标题
- 给一个打开的理由

## Content Quality Checks

定稿前检查：

1. **文字洁癖** — 语言干净吗？有没有 AI 味残留？
2. **表达效率** — 核心观点能一句话说清吗？有没有 99% 的文字在包装 1% 的内容？
3. **认知落差** — 同行把这事讲清楚了吗？你的表达比他们好在哪？
4. **读者价值** — 读者看完能带走什么？
5. **产品导向** — 这篇文章为谁服务？还是纯自嗨？

## What to Avoid

- 不要写「干货」内容——所有让你讲干货的建议都是不专业的
- 不要让 AI 分析爆款当方法论——规则大于结果
- 不要讨论「什么内容容易火」——火不等于赚钱
- 不要把你也不确定的论断写成铁板钉钉
- **不要保留任何「视频来源」的痕迹——文章就是文章**
